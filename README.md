# 🍕 PizzaCraft

A playful pizza restaurant website inspired by cozy cooking games.

PizzaCraft allows users to explore the menu, discover best-selling pizzas, read customer reviews, and build their own custom pizza through an interactive experience.

## ✨ Features

- 🎮 Game-inspired UI/UX
- 🍕 Interactive pizza menu
- ⭐ Best-selling products section
- 🧾 Customer testimonials
- 🛒 Pizza builder experience
- 📱 Responsive design
- 🎨 Custom illustrations and assets
- ⚡ Smooth animations with Framer Motion
